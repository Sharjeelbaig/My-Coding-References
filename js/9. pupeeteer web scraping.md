# Scrap google search results with Puppeteer
Puppeteer is a powerful Node.js library that provides a high-level API to control headless Chrome or Chromium browsers. It is widely used for web scraping, automated testing, and other browser automation tasks. In this guide, we will walk through the steps to scrape Google search results using Puppeteer.
## Jump instantly to code
```javascript

import puppeteer from "puppeteer";

async function searchGoogle(q) {
  const browser = await puppeteer.launch({ headless: true });
  const page = await browser.newPage();

  await page.goto(`https://www.google.com/search?q=${encodeURIComponent(q)}`, {
    waitUntil: "networkidle2"
  });

  const links = await page.evaluate(() =>
    [...document.querySelectorAll("a")]
      .map(a => a.href)
      .filter(h => h.startsWith("https://") && !h.includes("google.com"))
      .slice(0, 5)
  );

  await browser.close();
  return links;
}

console.log(await searchGoogle("who is sharjeel baig"));
```
