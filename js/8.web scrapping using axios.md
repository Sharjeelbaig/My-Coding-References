# Web Scraping using JavaScript
Web scraping is the process of extracting data from websites. JavaScript, being a versatile language, offers several libraries and tools to facilitate web scraping. Below are some popular methods and libraries for web scraping using JavaScript.

## 1. Using Node.js with Axios and Cheerio
One of the most common ways to scrape web pages is by using Node.js along with the Axios library for making HTTP requests and Cheerio for parsing HTML.
```javascript
import axios from 'axios';
import cheerio from 'cheerio';
const url = 'https://example.com';
axios.get(url)
  .then(response => {
    const html = response.data;
    const $ = cheerio.load(html);
    const data = [];
    $('selector').each((index, element) => {
      data.push($(element).text());
    });
    console.log(data);
  })
  .catch(error => {
    console.error('Error fetching the page:', error);
  });
```
